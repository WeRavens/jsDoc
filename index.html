Pengenalan JavaScript

JavaScript (JS) adalah bahasa pemrograman yang digunakan untuk membuat website menjadi interaktif, dinamis, dan hidup.

1. Peran JavaScript di Web

Jika kita mengibaratkan website sebagai manusia, maka:

  • HTML (Structure): Adalah Tulang/Kerangka (Menentukan isi dan struktur).
  • CSS (Style): Adalah Baju/Penampilan (Mengatur warna, font, dan tata letak).
  • JavaScript (Behavior): Adalah Otak/Otot (Menangani logika, klik tombol, animasi, dan data).
Fakta Penting: JavaScript TIDAK SAMA dengan Java. Keduanya adalah bahasa yang sangat berbeda, layaknya "Car" (Mobil) dan "Carpet" (Karpet).

2. Cara Menulis JavaScript

Kode JS dapat ditulis langsung di dalam file HTML menggunakan tag <script>, atau di file terpisah (External).

a. Internal (Di dalam HTML)

Biasanya diletakkan tepat sebelum penutup tag </body> agar website loading lebih cepat.

<!DOCTYPE html><html><body> <h1>Halo Dunia</h1> <!-- Kode JS ditulis di sini --> <script> alert("Selamat Datang!"); </script></body></html>

b. External (File Terpisah - Best Practice)

Digunakan untuk proyek profesional agar kode rapi.

<!-- Menghubungkan ke file script.js --><script src="script.js"></script>

3. Cara Menampilkan Output

Sebagai developer, kita sering menggunakan console untuk mengecek apakah kode berjalan benar.

console.log("Pesan ini muncul di Console browser");alert("Ini adalah notifikasi!");document.write("Teks ini ditulis oleh JavaScript");

4. Kenapa JavaScript Sangat Populer?

Saat ini JS adalah bahasa pemrograman paling populer di dunia karena:

  • Fullstack: Bisa jalan di Browser (Frontend) dan Server (Backend dengan Node.js).
  • Framework Besar: Memiliki ekosistem raksasa seperti React, Vue, dan Angular.
  • Komunitas: Sangat mudah mencari solusi jika terjadi error.

Variables & Data Types

Variabel adalah wadah untuk menyimpan data. Bayangkan variabel seperti sebuah kardus yang kita beri label nama agar isinya bisa kita cari dan gunakan nanti.

1. Deklarasi Variabel (Var, Let, Const)

Di JavaScript modern, ada tiga cara membuat variabel. Namun, cara penggunaannya berbeda.

a. var (Cara Lama / Legacy)

Dulu ini satu-satunya cara. Sekarang tidak disarankan karena memiliki masalah pada lingkup (scope) yang bisa menyebabkan bug.

var nama = "Budi";var nama = "Andi"; console.log(nama);
Hindari var! Gunakan hanya jika kamu memelihara kode legacy (kode tua). Untuk proyek baru, selalu gunakan let atau const.

b. let (Nilai Bisa Berubah)

Gunakan let jika nilai variabel tersebut akan berubah di masa depan (misalnya: skor game, counter, umur).

let skor = 0;skor = 10; skor = 20;

c. const (Nilai Tetap / Konstanta)

Gunakan const untuk nilai yang tidak boleh berubah selamanya (misalnya: ID User, Phi, Rumus pasti). Ini adalah default terbaik.

const phi = 3.14;console.log(phi);
Tips Profesional: Selalu gunakan const secara default. Hanya ganti ke let jika kamu yakin nilainya perlu diubah nanti.

2. Tipe Data Dasar (JS Types)

JavaScript memiliki tipe data dinamis (Dynamic Typing), artinya kita tidak perlu menulis tipe datanya secara manual, komputer akan menebaknya sendiri.

a. String (Teks)

Kumpulan karakter yang diapit oleh tanda kutip ("", '', atau backticks ``).

let namaDepan = "John"; let namaBelakang = 'Doe'; let sapaan = `Halo ${namaDepan}`;

b. Number (Angka)

Mencakup bilangan bulat (integer) dan desimal (float).

let umur = 25;let harga = 15000.50;let suhu = -5;

c. Boolean (Logika)

Hanya memiliki dua nilai: true (Benar) atau false (Salah). Sering dipakai untuk pengkondisian (If/Else).

let isOnline = true;let isGameOver = false;

d. Empty Values (Null & Undefined)

Dua tipe data spesial yang sering membingungkan pemula.

  • Undefined: Variabel sudah dibuat, tapi belum diisi apa-apa. (Kosong tidak sengaja).
  • Null: Variabel sengaja dikosongkan oleh programmer. (Kosong sengaja).
let kotakKosong; console.log(kotakKosong); let kotakSampah = null;console.log(kotakSampah);

3. Cek Tipe Data

Gunakan operator typeof untuk mengetahui tipe data sebuah variabel.

console.log(typeof "Halo"); console.log(typeof 100); console.log(typeof true);

Control Flow (Logika If-Else)

Control flow memungkinkan kode berjalan dinamis berdasarkan kondisi tertentu. Tanpa ini, kode hanya akan berjalan lurus dari atas ke bawah.

1. Konsep Dasar (If & Else)

Bayangkan sebuah pos keamanan. Kode di dalam blok if hanya akan dijalankan jika kondisinya True (Benar).

let isUserOnline = true;if (isUserOnline) { console.log("üü¢ User Online");} else { console.log("‚ö™ User Offline");}

2. Menangani Banyak Kondisi (Else If)

Gunakan else if ketika ada lebih dari dua kemungkinan. Komputer akan mengecek dari atas, dan berhenti pada kondisi pertama yang benar.

let lampu = "kuning";if (lampu === "merah") { console.log("‚õî Berhenti!");} else if (lampu === "kuning") { console.log("‚ö†Ô∏è Bersiap"); } else if (lampu === "hijau") { console.log("‚úÖ Jalan");} else { console.log("Lampu Rusak");}

3. Operator Perbandingan

Untuk menghasilkan nilai True/False, kita membutuhkan operator perbandingan.

  • >, <, >=, <= : Perbandingan angka standar.
  • === : Sama dengan (Strict). Cek Nilai DAN Tipe Data.
  • !== : Tidak sama dengan.
Hindari == (Loose Equality)!5 == "5" hasilnya True. Ini berbahaya karena mencampur Number dan String. Selalu gunakan ===.

4. Logika Ganda (&&, ||, !)

Digunakan untuk menggabungkan dua syarat atau lebih.

  • && (AND): Semua syarat harus benar.
  • || (OR): Salah satu syarat saja benar, hasil dianggap benar.
  • ! (NOT): Membalikkan nilai (True jadi False).
let umur = 20;let punyaTiket = true;if (umur >= 18 && punyaTiket === true) { console.log("Silakan Masuk");}

5. Truthy & Falsy Values

Di JavaScript, semua nilai dianggap True, kecuali 6 nilai Falsy ini:

  1. false
  2. 0 (Angka nol)
  3. "" (String kosong)
  4. null
  5. undefined
  6. NaN
let nama = ""; if (nama) { console.log("Halo " + nama);} else { console.log("Nama belum diisi!"); }

6. Best Practice: Guard Clauses

Hindari Nested If (If bersarang) yang terlalu dalam karena sulit dibaca. Gunakan teknik Guard Clauses (Cek error dulu, lalu return).

Pro Tip: Usir kondisi yang salah di awal fungsi agar kode utama tetap bersih. 
function login(user) { if (user) { if (user.password === "123") { return "Login Sukses"; } }}function loginPro(user) { if (!user) return "User tidak ditemukan"; if (user.password !== "123") return "Password Salah"; return "Login Sukses";}

JavaScript Functions

Panduan lengkap memahami fungsi, parameter, return value, hingga arrow function dalam JavaScript.

Pengantar

Function adalah blok kode yang dirancang untuk melakukan tugas tertentu (Reusable Code).

Contoh Kode Dasar

function sapa() { console.log("Halo Dunia!");}sapa();

1. Parameter dan Argument

Fungsi dapat menerima input agar lebih dinamis.

function jumlahkan(a, b) { return a + b;}console.log(jumlahkan(5, 10));
Ingat: Jangan lupa tanda kurung () saat memanggil fungsi!

2. Arrow Function

Sintaks modern dan lebih ringkas untuk menulis function.

function tambah(a, b) { return a + b;}const tambahArrow = (a, b) => a + b;console.log(tambahArrow(3, 7));
Best Practice: Gunakan arrow function untuk callback dan function sederhana.
